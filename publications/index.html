  1. ISBI
    From Specialist to Generalist: Unlocking SAM’s Learning Potential for Fetal Ultrasound Segmentation
    Vi Vu*, Hoang-Tan Nguyen*, and others

    Challenge Winner Abs Bib PDF Code

2025

  1. BMVC
    Semi-MoE: Mixture-of-Experts meets Semi-Supervised Histopathology Segmentation
    Vi Vu*, Tuan-Hung Nguyen*, and others
    In British Machine Vision Conference (BMVC), 2025
    Abs Bib PDF Code

    We propose Semi-MoE, a novel framework that leverages Mixture-of-Experts to address the homogeneity problem in semi-supervised histopathology segmentation. Our method introduces specialized expert networks that handle diverse domain distributions in pathology images, improving segmentation performance under limited annotations.

  2. CVPRW
    HDC: Hierarchical Distillation for Multi-level Noisy Consistency in Semi-supervised Histopathology Segmentation
    Vi Vu*, Tran-Quang-Khai Le*, and others
    In IEEE/CVF Conference on Computer Vision and Pattern Recognition Workshops (CVPRW), 2025
    Abs Bib PDF Code

    We introduce HDC, a hierarchical distillation framework that addresses multi-level noise in pseudo-labels for semi-supervised histopathology segmentation. Our approach employs a cascaded consistency regularization strategy that progressively refines predictions across multiple semantic levels.

  3. MICCAI-W
    Learning Disentangled Stain and Structural Representations for Robust Histopathology Analysis
    Vi Vu*, Huy-Hoang Pham*, and others
    In MICCAI Workshop on Computational Pathology (COMPAYL), 2025
    Abs Bib PDF Code

    We propose a mean-teacher framework for disentangling stain appearance from structural content in histopathology images. By separating H&E staining variations from underlying tissue morphology, our method achieves robust segmentation performance across diverse staining protocols with minimal labeled data.

  4. AAAI Bridge
    Semi-Supervised Histopathology Image Segmentation with Feature Diversified Collaborative Learning
    Vi Vu* and others
    In AAAI Bridge Program on AI for Medicine, 2025
    Abs Bib PDF Code

    We present a collaborative learning framework that promotes feature diversity in semi-supervised histopathology segmentation. By encouraging complementary representations across multiple learners, our method reduces confirmation bias and improves pseudo-label quality.
